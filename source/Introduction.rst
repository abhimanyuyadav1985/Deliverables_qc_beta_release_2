.. Deliverables QC classes

Introduction
==========================

A python PyQt based application running on windows system, connected to instrument room database and the DUG system.
The Database connection is powered by SQLAlchemy + psycopg2 engine. Multi-threaded with a SEGD, SEGY and Shipment modules.
Full integration with DUG tape server control onver Insight project structure for deliverables data storage. A variety of media and file formats are supported.
